Fonctionnalités

Tout ce qu'il faut pour gérer une agence

7 modules connectés entre eux, pensés pour les agences de location de voitures en Algérie.

Gestion de flotte

Suivez chaque véhicule : marque, modèle, immatriculation, kilométrage, statut (disponible / loué / maintenance), photo, prix journalier.

Fichier clients

Coordonnées, permis, historique de locations, total dépensé. Recherche instantanée. Statut actif / inactif.

Chauffeurs

Vos employés chauffeurs avec permis, expiration, tarif journalier, salaire mensuel. Assignables aux réservations.

Réservations

Création en quelques clics : client + véhicule + chauffeur + dates → prix calculé. Statuts à venir / actif / terminé. Disponibilité vérifiée.

Retours & dépassements

Saisissez kilométrage retour, carburant, état, dommages. Le système calcule automatiquement les frais kilométriques excédentaires.

Maintenance

Historique des interventions par véhicule. Rappels automatiques pour les prochaines maintenances (date ou kilométrage).

Contrats & Factures PDF

Génération automatique de contrats de location (8 articles légaux) et factures imprimables, au format A4.

Mode sombre & clair

Interface adaptable : mode sombre élégant ou mode clair épuré. Bilingue Français/Arabe avec RTL automatique.

Questions fréquentes

FAQ

L'application est-elle vraiment gratuite ?

Oui, AutoLoc est gratuit. L'application Windows est entièrement gratuite avec toutes les fonctionnalités. La version web propose un essai gratuit et un abonnement mensuel pour l'accès cloud.

Mes données sont-elles en sécurité ?

Avec l'application Windows : 100% de vos données restent sur votre ordinateur (base SQLite locale). Personne d'autre n'y a accès. Avec la version web : les données sont chiffrées et stockées sur des serveurs sécurisés européens.

Faut-il une connexion Internet ?

Non pour l'application Windows : elle fonctionne entièrement hors-ligne. Oui pour la version web (forcément). Vous pouvez utiliser les deux à la fois.

Combien d'utilisateurs / véhicules / clients je peux gérer ?

Aucune limite. Vous pouvez ajouter autant de véhicules, clients, chauffeurs, réservations que vous voulez.

L'application est-elle en arabe ?

Oui ! AutoLoc est bilingue Français / Arabe avec support RTL (droite-à-gauche) complet. Vous pouvez basculer entre les deux langues à tout moment.

Comment imprimer un contrat de location ?

Depuis la page Réservations, cliquez sur « Contrat » à côté d'une réservation. Un contrat de location au format A4 (avec 8 articles légaux) s'ouvre et peut être imprimé ou enregistré en PDF.
